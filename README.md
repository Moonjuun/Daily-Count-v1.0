# daily-count 데일리카운트

## Purpose

- 리액트를 이용해서 간단한 웹사이트를 만들어 보고 배포까지 해봄
- 도메인을 구매해서 AWS S3를 배포를 하고 https 적용까지 해봄

## Tech stack

`React`, `AWS`, `Pico css`, `React snap`

## Getting Started

#### Install

```
npm i
```

#### Build

```
npm run build
```

#### Start

```
npm start
```

## Site

https://daily-count.com
